## 🚀 Quick Wins - Start Here

### If you scored less than 6 in any section:

**Administrative:**
- [ ] Designate your HIPAA compliance officer today
- [ ] Download a risk analysis template
- [ ] Schedule annual security training

**Technical:**
- [ ] Enable automatic screen lock (5 min)
- [ ] Turn on full-disk encryption (BitLocker/FileVault)
- [ ] Enable multi-factor authentication

**Physical:**
- [ ] Install access control on exterior doors
- [ ] Create a visitor sign-in log
- [ ] Use a certified e-waste vendor for disposal
