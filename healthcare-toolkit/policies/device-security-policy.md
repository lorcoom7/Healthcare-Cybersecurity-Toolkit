# Device Security Policy

## Healthcare Organization: _____________________________

---

## 1. Purpose

This Device Security Policy establishes requirements for securing all devices that access, store, or transmit Protected Health Information (PHI) in compliance with HIPAA Security Rule §164.310(d).

## 2. Scope

This policy applies to:
- Corporate workstations and laptops
- Mobile devices (smartphones, tablets)
- Medical devices
- Removable media
- Network devices
- Cloud-connected devices

---

## 3. Device Classification

| Classification | Description | Examples |
|----------------|-------------|----------|
| **High** | Stores/accesses PHI directly | EHR workstations, clinical laptops |
| **Medium** | Business use with email | Administrative laptops |
| **Low** | Limited access | Guest devices, demonstration equipment |

## 6. Removable Media

### 6.1 Acceptable Use

| # | Guideline | Agreed |
|---|-----------|--------|
| 1 | Only organization-issued USB devices | ☐ |
| 2 | No personal removable media | ☐ |
| 3 | Encrypt all data on USB devices | ☐ |
| 4 | Scan for malware before use | ☐ |
| 5 | Return devices to IT for disposal | ☐ |

### 6.2 Prohibited Actions

- ☐ Connecting unknown USB devices
- ☐ Using personal USB drives
- ☐ Storing PHI on unencrypted media
- ☐ Leaving devices unattended with data
- ☐ Disposing of media without wiping
